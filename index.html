VerIDial.Workshop

Sample code for VerIDial Workshop Norwich

This code is written in C# using MVC. The initial page will allow the user to enter a number and pick a PIN transmission method from voice, SMS or app. The system will call the VerIDial REST API to start the process and then switch over to an iframe plug-in for the entry of the PIN. Lastly, there are two pages, one for success, one for failure that the user will get redirected to after submission. These pages will call the VerIDial REST API to retrieve the status textual description and display on screen.

Please note that this is not production code. There is no logging, very little error handling and has undergone no code review. Use it as an example of how VerIDial (and 2FA in general) fits into the flow of the system(s) you are designing.
